Type	        Stores Data For	                                             Data Persists After Closing Browser?

localStorage.clear();

sessionStorage.setItem("user", "Dunna");

let name = sessionStorage.getItem("user");

Remove Specific Item
sessionStorage.removeItem("user");

Clear All
sessionStorage.clear();





7) scope :
Types of Scope in JavaScript:

Type	                    Description
Global Scope	            Variables declared outside any function or block are global and can be accessed anywhere in the program.
Local (Function) Scope	    Variables declared inside a function are local to that function and can’t be accessed outside.
Block Scope	            Variables declared inside {} (using let or const) are only accessible within that block.

global scope :

let name = "Sumalatha"; // Global variable

function greet() {
  console.log("Hello " + name); // Can access global variable
}

greet(); // Works
console.log(name); //  Works

Local (Functional) Scope:

function showMessage() {
  let message = "Hello World"; // Local variable
  console.log(message);
}

showMessage(); // Works
console.log(message); //  Error (not accessible outside function)


Block Scope : (with let and const)
{
  let x = 10;
  const y = 20;
  var z = 30;
}

console.log(z); // Works (var is not block-scoped)
console.log(x); // Error (let is block-scoped)
console.log(y); // Error (const is block-scoped)




8) JS Objects:

A JavaScript object is a collection of key-value pairs.
It is used to store multiple values together — like a mini database of related information.

Syntax:

let objectName = {
  key1: value1,
  key2: value2,
  key3: value3
};


Each key is called a property, and each value can be anything — a string, number, array, another object, or even a function.
Example:

let person = {
  name: "Sumalatha",
  age: 25,
  city: "Hyderabad"
};


Here:

name, age, and city are keys (properties)

"Sumalatha", 25, "Hyderabad" are values

Accessing Object Values:

There are two ways to access values:

Dot notation

console.log(person.name);  // Output: Sumalatha

Bracket notation

console.log(person["city"]);  // Output: Hyderabad



9)Destructuring :

Destructuring means unpacking values from arrays or objects into separate variables easily.

Instead of extracting values one by one, destructuring lets you do it in a single line.

1. Array Destructuring Example:

let colors = ["red", "green", "blue"];
let [first, second, third] = colors;

console.log(first);   // red
console.log(second);  // green
console.log(third);   // blue

The values from the array are automatically assigned to variables in the same order.




10) JavaScript Loops:


1. for Loop

Used when you know how many times you want to run the code.

Syntax:
for (initialization; condition; increment/decrement) {
  // code to run
}

Example:
for (let i = 1; i <= 5; i++) {
  console.log("Number:", i);
}


Output:

Number: 1  
Number: 2  
Number: 3  
Number: 4  
Number: 5

2. while Loop

Used when you don’t know in advance how many times you need to loop.

Syntax:
while (condition) {
  // code to run
}

Example:
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}

3. do...while Loop

This loop runs the code at least once, even if the condition is false.

Syntax:
do {
  // code to run
} while (condition);

Example:
let i = 1;
do {
  console.log(i);
  i++;
} while (i <= 3);

Output:

1  
2  
3

4. Nested Loops

Loop inside another loop.

Example:
for (let i = 1; i <= 3; i++) {
  for (let j = 1; j <= 2; j++) {
    console.log(`i = ${i}, j = ${j}`);
  }
}

Output:

i = 1, j = 1  
i = 1, j = 2  
i = 2, j = 1  
i = 2, j = 2  
i = 3, j = 1  
i = 3, j = 2

13) Get elements in js:

In JavaScript, to access or change something on a webpage (like a heading, paragraph, image, or input),
you first need to get (select) that element from the DOM (Document Object Model).

JavaScript gives us several methods to do this 

1. document.getElementById()
Description:

Finds one single element by its id attribute.

Example:
<h2 id="title">Hello!</h2>

<script>
let heading = document.getElementById("title");
heading.style.color = "blue";
</script>


Explanation:

getElementById("title") finds the element with id="title".

Then we changed its color.

Returns: a single element (or null if not found)

2. document.getElementsByClassName()
Description:

Finds all elements with a given class name.

Example:
<p class="note">First note</p>
<p class="note">Second note</p>

<script>
let notes = document.getElementsByClassName("note");
notes[0].style.color = "red";  // first <p>
notes[1].style.color = "green"; // second <p>
</script>


Explanation:

Returns a collection (array-like list) of all elements having the same class.

You can access them using [index].

Returns: HTMLCollection

3. document.getElementsByTagName()
Description:

Finds all elements with a specific HTML tag.

Example:
<h3>Heading 1</h3>
<h3>Heading 2</h3>

<script>
let headings = document.getElementsByTagName("h3");
for (let i = 0; i < headings.length; i++) {
  headings[i].style.color = "purple";
}
</script>


Explanation:

Selects all <h3> tags.

Loops through them to change color.

Returns: HTMLCollection

4. document.querySelector()
Description:

Finds the first element that matches a CSS selector.

Example:
<p class="msg">Hello</p>
<p class="msg">Hi</p>

<script>
let firstPara = document.querySelector(".msg");
firstPara.style.color = "blue";
</script>


Explanation:

.msg is a CSS selector (selects by class name).

It changes only the first matching element.

Returns: single element (first match)

5. document.querySelectorAll()
Description:

Finds all elements that match a CSS selector.

Example:
<p class="text">Para 1</p>
<p class="text">Para 2</p>

<script>
let paras = document.querySelectorAll(".text");
paras.forEach(p => p.style.color = "orange");
</script>


Explanation:

.text selects all elements with class text.

Loops through using forEach() to apply changes.

Returns: NodeList (can use forEach())

6. document.getElementsByName()
Description:

Finds elements with a specific name attribute (often used with forms).

Example:
<input type="text" name="username" value="Sumalatha">
<input type="text" name="username" value="Anil">

<script>
let inputs = document.getElementsByName("username");
console.log(inputs[0].value); // Output: Sumalatha
</script>

Returns: NodeList

Merging Arrays in JS:

Merging (or combining) arrays means joining two or more arrays into a single array.

Example:

let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let merged = [1, 2, 3, 4, 5, 6];

Method 1: Using concat() (Easiest Way)

The concat() method joins two or more arrays and returns a new array (it doesn’t change the originals).

Example:
let arr1 = [10, 20, 30];
let arr2 = [40, 50, 60];

let merged = arr1.concat(arr2);
console.log(merged);


Output:

[10, 20, 30, 40, 50, 60]


You can also merge more than two:

let arr3 = [70, 80];
let all = arr1.concat(arr2, arr3);
console.log(all); // [10, 20, 30, 40, 50, 60, 70, 80]

Method 2: Using Spread Operator (...)

The spread operator is the modern, shorter way to merge arrays (ES6+).

Example:
let arr1 = ["A", "B", "C"];
let arr2 = ["D", "E", "F"];

let merged = [...arr1, ...arr2];
console.log(merged);


Output:

["A", "B", "C", "D", "E", "F"]


You can also add new elements while merging:

let final = [...arr1, "X", ...arr2, "Y"];
console.log(final); // ["A", "B", "C", "X", "D", "E", "F", "Y"]





18) Shifting Elements :

In JavaScript, shifting elements means removing or adding elements at the beginning of an array.

There are two main array methods related to shifting:

Method	           Action	        Position	Description
shift()	          Removes first element	Beginning	Moves all other elements down
unshift()	  Adds new elements	Beginning	Moves existing elements up
1. shift() Method
Syntax:
array.shift();

Example:
let fruits = ["apple", "banana", "cherry"];
let removed = fruits.shift();

console.log(fruits);  // Output: ["banana", "cherry"]
console.log(removed); // Output: apple


Explanation:

The first element (apple) is removed.

Remaining elements shift down one position.

It returns the removed element.

*shift() Important Points:

Changes (mutates) the original array 

Returns the removed element 

New array starts from the next element

2. unshift() Method
Syntax:
array.unshift(newElement1, newElement2, ...);

Example:
let colors = ["blue", "green"];
colors.unshift("red");

console.log(colors); // Output: ["red", "blue", "green"]

Explanation:

"red" is added to the beginning of the array.

All other elements are shifted right.

It returns the new length of the array.

20) Conditional statements :


Conditional statements are used to make decisions in JavaScript.
They let your program run certain code only if a specific condition is true.

Types of Conditional Statements in JavaScript

1. if statement

Used when you want to execute code only if a condition is true.

let age = 18;

if (age >= 18) {
  console.log("You are an adult.");
}

Output:

You are an adult.


2. if...else statement

Used when you want to execute one block of code if the condition is true,
and another block if it’s false.

let age = 15;

if (age >= 18) {
  console.log("You can vote.");
} else {
  console.log("You cannot vote.");
}

Output:

You cannot vote.


3. if...else if...else statement

Used when you have multiple conditions to test.

let marks = 75;

if (marks >= 90) {
  console.log("Grade: A");
} else if (marks >= 70) {
  console.log("Grade: B");
} else if (marks >= 50) {
  console.log("Grade: C");
} else {
  console.log("Fail");
}

Output:

Grade: B


4. switch statement

Used when you want to test one value against many possible cases.

let day = 3;

switch (day) {
  case 1:
    console.log("Monday");
    break;
  case 2:
    console.log("Tuesday");
    break;
  case 3:
    console.log("Wednesday");
    break;
  case 4:
    console.log("Thursday");
    break;
  default:
    console.log("Invalid day");
}

 Output:

Wednesday

break stops the switch from checking the next cases.
default runs if none of the cases match.


5. Ternary Operator (? :)

A short way to write simple if...else statements.

let age = 20;
let message = (age >= 18) ? "Adult" : "Minor";
console.log(message);


Output:

Adult
